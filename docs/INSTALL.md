# Installation

### Requirements
All the codes are tested in the following environment:
* Linux (tested on Ubuntu 18.04)
* Python 3.8
* PyTorch 1.7.1
* CUDA 10.2
* [spconv v2.x](https://github.com/traveller59/spconv)


### Install OpenSeg3D

a. Clone this repository.
```shell
git clone https://github.com/WPCLab/OpenSeg3D
```
